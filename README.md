# 拱北案管平台

# 项目构建
```
# 安装项目依赖包
cnpm install

# 启动自动刷新
cnpm run watch
```

## 目录结构
+ leader 所领导
+ ephor 探长
+ police 民警
+ examine 案审大队
+ attache 案管专员
+ logistics 内勤

+ caseControl.html 案件监管
+ myFollow.html 我的关注
+ cases.html 我的案件
+ caseDetails.html 案件详情
+ visitList.html 来访登记列表
+ visitEdit.html 来访登记编辑
+ caseMyVisit.html 我的来访
+ caseConfirmList.html 案件确认列表
